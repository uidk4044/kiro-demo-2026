# Kiro AI Demo App

这是一个由 Kiro AI 自动创建的演示应用，展示了 AI 驱动的自动化开发工作流。

## 特性

- 🤖 完全由 AI 生成的代码
- 🚀 自动推送到 GitHub
- 🎨 现代化的 UI 设计
- 📡 RESTful API
- ⚡ Express.js 后端

## 安装

```bash
npm install
```

## 运行

```bash
npm start
```

然后访问 http://localhost:3000

## 技术栈

- Node.js
- Express.js
- HTML/CSS/JavaScript

## 关于 Kiro

Kiro 是一个 AI 驱动的 IDE，可以：
- 自动生成代码
- 直接操作 GitHub
- 执行完整的开发工作流
- 通过 MCP 协议连接各种服务

这个项目展示了 Kiro 如何在几秒钟内创建一个完整的 Web 应用并推送到 GitHub。
